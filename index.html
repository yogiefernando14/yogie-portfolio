Luxury Streetwear E-Commerce Website

Role

Frontend Developer & UI Designer

Project Overview

YOGIEZ is a luxury streetwear e-commerce website designed and developed from scratch using HTML, CSS and Vanilla JavaScript. The project focuses on delivering a premium shopping experience with a minimalist luxury aesthetic and mobile-first design approach.

Key Features

  • Responsive Mobile-First Design
  • Luxury Black & White Branding
  • Product Catalog System
  • Dedicated Product Pages
  • Product Detail Experience
  • Size Selection System
  • Quantity Selector
  • Shopping Cart Functionality
  • Dynamic Cart Counter
  • Automatic Total Price Calculation
  • Local Storage Integration
  • Lookbook Section
  • Sticky Navigation
  • Mobile Friendly UI

Technologies Used

  • HTML5
  • CSS3
  • Vanilla JavaScript
  • Local Storage API
  • GitHub Pages

Project Highlights

  • Persistent cart storage
  • Scalable architecture for future collections
  • Performance-focused implementation
  • Luxury-inspired user interface
